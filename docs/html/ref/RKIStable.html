RKIStable

RKIStable[rk]

returns an algebraic condition equivalent to rk being stable on the imaginary axis.

Details and Options

  • A Runge–Kutta method is I-stable if for all , where is the linear stability function.
  • RKIStable returns True or False if rk does not contain free variables. Otherwise, it will return an (in)equality dependent on the free variables.
  • The following options can be given:
  • EmbeddedFalsewhether to use the embedded coefficients
    StageNonetreat a stage as the solution
    DenseOutputFalsehow to evaluate dense output
